CSS Properties Cheatsheat

Name Description Values
Opacity Changes the opacity 0 to 1 or a %
Font-family Sets the typeface for the given element Abundance of web-safe fonts available
Color Sets the foreground color Named, RGB, HSL, or Hexadecimal
Background Color Sets the backgroundcolor Named, RGB, HSL, or Hexadecimal
Border Places a border around the given element Takes 3 values: weight, style, color
Font-style Allows you to italicize text italic or normal
Font-weight Allows you to bold text Four values: bold, normal, bolder, stronger, or a number between 1 and 1000
